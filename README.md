
# Wireshark Packet Analysis Lab

## Objective
To capture and analyze network packets to understand network communication.

## Tools Used
- Kali Linux
- Wireshark

## Protocols Analyzed
- DNS
- TCP
- ICMPv6

## Activity Performed
- Started packet capture
- Opened a website
- Filtered DNS traffic
- Inspected packets

## Findings
- Observed DNS queries
- Tracked TCP handshake
- Observed ICMPv6 packets and neighbor discovery messages

## Skills Learned
- Packet capture
- Protocol analysis
- Traffic filtering
